Minecraft Server Tool
How to create a server
-------- Step 1 --------
Either:
-
Press the “Create New Server” button
-
Select a file with the “Select/drop .jar” option (you can also drop a .jar file
directly onto the panel from file explorer)
-------- Step 2 --------
Create New Server
1. Make sure the “Server” tab is selected
2. Create a name for your server*
3. Select a version for your server
4. Press “Create”
Select/drop .jar
1. Create a name for your server*
2. Press “Create”
NOTE: Server names can only include letters, numbers and spaces
-------- Step 3 --------
If you selected Create New Server and you don’t select the latest version or 1.8 then
the server file has to be downloaded from the internet.
Actions the Minecraft Server Tool is doing:
-
Downloading the .jar and the official Minecraft Server Tool plugin
-
Creating a folder in the Servers folder
-
Creating a file to store data about the server (server.json)
-
Creating the startup script (server.bat)
-
Starting the server
The last step is accepting the EULA. Once you do that the server will be created and
ready for use!
