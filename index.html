About

The Minecraft Server Tool is a program designed to help Minecraft players set up multiplayer servers on their own computer and monitor them easily. The Minecraft Server Tool comes with built in management solutions for server networks and complex proxy chains.

Features

  • Player manager
  • World manager
  • Console
  • Scheduler
  • Automated backups
  • Performance monitor
  • Powerful control panel
  • Custom action options
  • Full customisation and settings
  • Update support
  • File editor
  • Downloads configuration
  • Plugins library
  • Mods library
  • World configuration
  • Datapack library
